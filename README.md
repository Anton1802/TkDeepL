# TkDeepL
This is a simple client on Linux, for translator DeepL.
## Advantages
- Simple
- Without API
## Dependencies
- Tkinter
- Playwright
## Installing
### Step 1: 
```
sudo apt-get install python3-tk
```
### Step 2:
```
pip install pytest-playwright
playwright install
```
## Run
```
python3 main.py
```
